# 6.2 LEVEL TURN

The flight path and forces for an airplane in a level turn are sketched in Fig. 6.1. Here, the flight path is *curved*, in contrast to the rectilinear motion studied in Chapter 5. By definition, a *level turn* is one in which the curved flight path is in a horizontal plane parallel to the plane of the ground; that is, in a level turn the altitude remains constant.
