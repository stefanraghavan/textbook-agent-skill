Using the values of $C_{D,0}$ from Fig. 5.23, we calculate the drag at several Mach numbers.

Assume $M_\infty = 0.9$. Then $C_{D,0} = 0.024$ (from Fig. 5.23). At 30,000 ft, $a_\infty = 994.7$ ft/s.

$$
V_\infty = 0.9(994.7) = 895.23 \ \text{ft/s}
$$

$$
q_\infty = \frac{1}{2}\rho_\infty V_\infty^2 = \frac{1}{2}(8.9068 \times 10^{-4})(895.23)^2 = 356.9 \ \text{lb/ft}^2
$$

To treat the drag due to lift, we need the value of $C_L$, obtained from

$$
C_L = \frac{W}{q_\infty S} = \frac{73{,}000}{(356.9)(950)} = 0.215
$$

From Eq. (2.47)

$$
C_D = C_{D,0} + KC_L^2 = 0.024 + (0.08)(0.215)^2 = 0.0277
$$

$$
D = q_\infty S C_D = (356.9)(950)(0.02777) = 9{,}392 \ \text{lb}
$$

This value is denoted by point 1 in Fig. 5.24, which is a plot of drag in pounds as a function of $M_\infty$.
