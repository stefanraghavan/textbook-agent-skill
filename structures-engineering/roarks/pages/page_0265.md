## TABLE 8.13 Collapse loads with plastic hinge locations for straight beams

NOTATION: $M_p =$ fully plastic bending moment (force-length); $x_h =$ position of a plastic hinge (length); $W_c =$ concentrated load necessary to produce plastic collapse of the beam (force); $w_c =$ unit load necessary to produce plastic collapse of the beam (force per unit length); $M_{oc} =$ applied couple necessary to produce plastic collapse (force-length). The fully plastic bending moment $M_p$ is the product of the yield strength of the material $\sigma_{ys}$ and the plastic section modulus $Z$ found in Table A.1 for the given cross sections

| Reference no., end restraints | Collapse loads with plastic hinge locations |
|---|---|
| 1a. Left end free, right end fixed (cantilever)  | $W_c = \dfrac{M_p}{l-a}$  |
| ![Figure 8.13-1a: Left end free, right end fixed (cantilever)](figures/fig8_13_1a.png) <!-- Description: Beam diagram showing a cantilever with left end free, right end fixed, a concentrated load W at distance a from the left end. --> | $x_h = l$ |
| 1b. Left end guided, right end fixed | $W_c = \dfrac{2M_p}{l-a}$ |
| ![Figure 8.13-1b: Left end guided, right end fixed](figures/fig8_13_1b.png) <!-- Description: Beam diagram showing left end guided, right end fixed, with concentrated load W at distance a from the left end. --> | $0 \leq x_{h1} \leq a \qquad x_{h2} = l$ |
| 1c. Left end simply supported, right end fixed | $W_c = \dfrac{M_p(l+a)}{a(l-a)}$ |
| ![Figure 8.13-1c: Left end simply supported, right end fixed](figures/fig8_13_1c.png) <!-- Description: Beam diagram showing left end simply supported, right end fixed, with concentrated load W at distance a from the left end. --> | $x_{h1} = a \qquad x_{h2} = l$ |
| 1d. Left end fixed, right end fixed | $W_c = \dfrac{2M_p l}{a(l-a)}$ |
| ![Figure 8.13-1d: Left end fixed, right end fixed](figures/fig8_13_1d.png) <!-- Description: Beam diagram showing both ends fixed, with concentrated load W at distance a from the left end. --> | $x_{h1} = 0 \qquad x_{h2} = a \qquad x_{h3} = l$ |
| 1e. Left end simply supported, right end simply supported | $W_c = \dfrac{M_p l}{a(l-a)}$ |
| ![Figure 8.13-1e: Left end simply supported, right end simply supported](figures/fig8_13_1e.png) <!-- Description: Beam diagram showing both ends simply supported, with concentrated load W at distance a from the left end. --> | $x_h = a$ |
| 1f. Left end guided, right end simply supported | $W_c = \dfrac{M_p}{l-a}$ |
| ![Figure 8.13-1f: Left end guided, right end simply supported](figures/fig8_13_1f.png) <!-- Description: Beam diagram showing left end guided, right end simply supported, with concentrated load W at distance a from the left end. --> | $0 < x_h < a$ |
