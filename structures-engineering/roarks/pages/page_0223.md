# TABLE 8.5 Shear, moment, slope, and deflection formulas for finite-length beams on elastic foundations *(Continued)*

**5. Externally created concentrated angular displacement**

![Figure 8.5: Externally created concentrated angular displacement](figures/fig8_5.png)
<!-- Description: Beam on elastic foundation with left-end reactions $M_A$, $R_A$, displacement $y_A$, rotation $\theta_A$, an imposed angular displacement $\theta_o$ near the right portion, and dimensions $a$ and $l$. -->

Transverse shear $= V = R_A F_1 - y_A 2EI\beta^3 F_2 - \theta_A 2EI\beta^2 F_3 - M_A \beta F_4 - \theta_o 2EI\beta^2 F_{\alpha 3}$

Bending moment $= M = M_A F_1 + \dfrac{R_A}{2\beta} F_2 - y_A 2EI\beta^2 F_3 - \theta_A EI\beta F_4 - \theta_o EI\beta F_{\alpha 4}$

Slope $= \theta = \theta_A F_1 + \dfrac{M_A}{2EI\beta} F_2 + \dfrac{R_A}{2EI\beta^2} F_3 - y_A \beta F_4 + \theta_o F_{\alpha 1}$

Deflection $= y = y_A F_1 + \dfrac{\theta_A}{2\beta} F_2 + \dfrac{M_A}{2EI\beta^2} F_3 + \dfrac{R_A}{4EI\beta^3} F_4 + \dfrac{\theta_o}{2\beta} F_{\alpha 2}$
