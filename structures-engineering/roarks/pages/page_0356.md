Transverse shear $= V = V_A - W\langle x-\theta\rangle^0$

Bending moment $= M = V_A R\sin x + M_A\cos x - T_A\sin x - WR\sin(x-\theta)\langle x-\theta\rangle^0$

Twisting moment $= T = V_A R(1-\cos x) + M_A\sin x + T_A\cos x - WR[1-\cos(x-\theta)]\langle x-\theta\rangle^0$

Deflection $= y = y_A + \Theta_A R\sin x + \psi_A R(1-\cos x) + \frac{M_A R^2}{EI}F_1 + \frac{T_A R^2}{EI}F_2 + \frac{V_A R^3}{EI}F_3 - \frac{WR^3}{EI}F_{a3}$

Bending slope $= \Theta = \Theta_A\cos x + \psi_A\sin x + \frac{M_A R}{EI}F_4 + \frac{T_A R}{EI}F_5 + \frac{V_A R^2}{EI}F_6 - \frac{WR^2}{EI}F_{a6}$

Roll slope $= \psi = \psi_A\cos x - \Theta_A\sin x + \frac{M_A R}{EI}F_7 + \frac{T_A R}{EI}F_8 + \frac{V_A R^2}{EI}F_9 - \frac{WR^2}{EI}F_{a9}$
