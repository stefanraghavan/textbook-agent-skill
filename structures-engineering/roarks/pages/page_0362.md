Transverse shear $= V = V_A$

Bending moment $= M = V_A R \sin x + M_A \cos x - T_A \sin x + M_o \cos (x-\theta)(x-\theta)^0$

Twisting moment $= T = V_A R(1-\cos x) + M_A \sin x + T_A \cos x + M_o \sin (x-\theta)(x-\theta)^0$

Vertical deflection $= y = y_A + \Theta_A R \sin x + \psi_A R(1-\cos x) + \dfrac{M_A R^2}{EI}F_1 + \dfrac{T_A R^2}{EI}F_2 + \dfrac{V_A R^3}{EI}F_3 + \dfrac{M_o R^2}{EI}F_{a1}$

Bending slope $= \Theta = \Theta_A \cos x + \psi_A \sin x + \dfrac{M_A R}{EI}F_4 + \dfrac{T_A R}{EI}F_5 + \dfrac{V_A R^2}{EI}F_6 + \dfrac{M_o R}{EI}F_{a4}$

Roll slope $= \psi = \psi_A \cos x - \Theta_A \sin x + \dfrac{M_A R}{EI}F_7 + \dfrac{T_A R}{EI}F_8 + \dfrac{V_A R^2}{EI}F_9 + \dfrac{M_o R}{EI}F_{a7}$

For tabulated values $V = K_V \dfrac{M_o}{R}, \quad M = K_M M_o, \quad T = K_T M_o, \quad y = K_y \dfrac{M_o R^2}{EI}, \quad \Theta = K_\Theta \dfrac{M_o R}{EI}, \quad \psi = K_\psi \dfrac{M_o R}{EI}$
