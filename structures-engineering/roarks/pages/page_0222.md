# TABLE 8.5 Shear, moment, slope, and deflection formulas for finite-length beams on elastic foundations *(Continued)*

**4. Concentrated intermediate moment**

![Figure 8.5: Concentrated intermediate moment](figures/fig8_5.png)
<!-- Description: Beam on elastic foundation with left-end quantities $M_A$, $y_A$, $R_A$, slope $\theta_A$, an intermediate applied moment $M_0$, distance $a$ from the left end to the moment, remaining length $l$, and coordinate axes $Y$ and $X$. -->

Transverse shear $= V = R_A F_1 - y_A 2EI\beta^3 F_2 - \theta_A 2EI\beta^2 F_3 - M_A \beta F_4 - M_0 \beta F_{a4}$

Bending moment $= M = M_A F_1 + \dfrac{R_A}{2\beta} F_2 - y_A 2EI\beta^2 F_3 - \theta_A EI\beta F_4 + M_0 F_{a1}$

Slope $= \theta = \theta_A F_1 + \dfrac{M_A}{2EI\beta} F_2 + \dfrac{R_A}{2EI\beta^2} F_3 - y_A \beta F_4 + \dfrac{M_0}{2EI\beta} F_{a2}$

Deflection $= y = y_A F_1 + \dfrac{\theta_A}{2\beta} F_2 + \dfrac{M_A}{2EI\beta^2} F_3 + \dfrac{R_A}{4EI\beta^3} F_4 + \dfrac{M_0}{2EI\beta^2} F_{a3}$

If $\beta l > 6$, see Table 8.6

Expressions for $R_A$, $M_A$, $\theta_A$, and $y_A$ are found below for several combinations of end restraints
